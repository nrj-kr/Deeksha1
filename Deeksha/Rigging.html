Rigging Questions

  1. What should you expect in a wire rope?

    1. Core lubrication.
    2. Dressing lubricants.
    3. Check for change in size of rope diameter.
    4. Colour of surface.
      5.

      C. Check for change in size of rope diameter.

  2. The length of tail rope in safety belt is:

    1. 2ft.
    2. 4ft.
    3. 5ft.
    4. 6ft.
      5.

      D. The length of tail rope in safety belt is 6ft or 1.8m.

  3. Above what height from ground, we should use Full Body Harness ?

    1. 2m.
    2. 2.5m.
    3. 3m
    4. 3ft.
      5.

      A. 2m.

  4. Diameter of Tail rope is?

    1. 1 inch.
    2. 2 inch.
    3. 3 inch.
    4. 4 inch.
      5.

      A. 1 inch.

  5. While working on a portable ladder, we should leave __ steps from top so as to use it as handle?

    1. 1 steps.
    2. 3 steps.
    3. 2 steps.
    4. 0 steps.
      5.

      B. 3 steps.

  6. Which of the following is a pulley block?

    1. Rope Pulley Block.
    2. Chain Pulley Block.
    3. Both
    4. None of the above.
      5.

      C. Both.

  7. Mechanical Advantage of Free pulley is

    1. 0.5
    2. 1.
    3. 1.5
    4. 2
      5.

      D. 2

  8. Mechanical Advantage is equal to the number of ___ running to the free pulley.

    1. Strands.
    2. Sheaves.
    3. Block
    4. Tackle.
      5.

      A. Strands.

  9. Which of the following knots is used by fire fighters ?

    1. Square Reef knot.
    2. Emergency knot.
    3. Bowline knot.
    4. Timber knot.
      5.

      B. Emergency knot.

  10. Knot used to lift any item vertically upwards is known as:

    1. Square reef knot.
    2. Emergency knot.
    3. Bowline knot.
    4. Timber knot.
      5.

      D. Timber knot.

  11. Wire ropes are made from.

    1. Copper.
    2. Manila rope.
    3. Steel
    4. Aluminium.
      5.

      C. Steel.

  12. Wire rope must be discarded if the reduction in the rope wire for general purpose rope is more than ?

    1. 11%.
    2. 10%.
    3. 6%.
    4. 7%.
      5.

      D. 7%.

  13. Which of the following are/is potential hazards of sling ?.

    1. Kinking.
    2. Birdcaging.
    3. Overloading
    4. All the above.
      5.

      D. All the above.

  14. Which of the following are/is a type of hitch?.

    1. Choker hitch.
    2. Vertical hitch.
    3. None of the above
    4. All the above.
      5.

      D. All the above.

  15. Which of the following are/is a portable lifting device?.

    1. Hydraulic Jack.
    2. mechanical Jack.
    3. Pneumatic Jack.
    4. All the above.
      5.

      D. All the above.

  16. Pascal's law is used in which of the following devices?.

    1. Chain Block.
    2. Jack.
    3. Puller
    4. All the above.
      5.

      B. Jack.
